# 📈 股票資訊整合平台

本專案旨在建立一個**股票的股價與成交量收集系統**，結合 **視覺化圖表繪製**、**筆記記錄**、**新聞摘錄**以及**研究報告整合**功能，協助使用者全面掌握股票資訊，輔助投資決策。

---

## 📌 專案特色

- ✅ 自動化爬取多檔股票每日價格與成交量  
- 📊 提供歷史數據視覺化圖表（如 K 線圖、成交量）  
- 📝 可針對個股撰寫個人筆記  
- 🗞 整合相關新聞摘錄，掌握市場脈動  
- 📄 整理研究報告以利資料分析
